Ik had een tool gevonden voor de game Eco dat handig was om je in-game dieet te optimaliseren. Maar ik merkte dat de UI heel basic was en dat de UX redelijk slecht was. In de README zag ik dat de maintainer nog zocht naar iemand om het design te verbeteren.

Daarom heb ik een Figma design gemaakt, getoond aan de maintainer, die het heel goed vond, en daarna heb ik het in code geïmplementeerd en de hele UI verbeterd en een pull request gedaan.
