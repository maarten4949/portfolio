Dit is een project dat ik heb gemaakt voor mijn eerste jaar MCT. Het was een opdracht om een hardware toestel te maken dat gegevens verzamelt en opslaat die weergegeven kunnen worden op een website.

De backend is gemaakt in FastAPI (Python) die gekoppeld is met een MySQL database. De frontend is gemaakt in Vanilla JS. De website wordt gehost op een Raspberry Pi (via Apache en Uvicorn). Ik heb ook verschillende sensoren gebruikt zoals: Time Of Flight, Ultrasone sensor en RFID sensor.
